systemli.org-Calc is an application which enables to work on a spreadsheet with multiple users. Thus you can create a spreadsheet or a shift schedule/rota with your friends. It is apparent which person wrote which content. All common spreadsheet functions are integrated in the application.

Features:

  • work on a spreadsheet together
  • a lot of common spreadsheet functions (sum, average...)
  • Import of spreadsheet documents

Obviously we make sure that all done entries cannot be intercepted. Therefore we are consequently using an encrypted connection to all spreadsheets.

All pads which have not been processed for 30 days will be automatically deleted!
